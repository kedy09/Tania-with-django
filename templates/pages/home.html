A Django starter project By Tania.

Ce projet Django est conçu comme un point de départ complet pour le développement web. Il utilise la dernière version de Django (5.0) et Python (3.12) pour offrir une base solide et moderne.

Le contenu actuel du projet comprend :

  • Gestion de la connexion, de l'inscription et de la réinitialisation de mot de passe des utilisateurs via l'intégration de django-allauth.
  • Configuration des fichiers statiques avec Whitenoise pour une meilleure performance.
  • Styling en utilisant le framework Bootstrap v5 pour une interface utilisateur élégante et responsive.
  • Débogage avancé grâce à l'intégration de django-debug-toolbar.
  • Formulaires simplifiés et réutilisables avec django-crispy-forms.
  • Pages d'erreur personnalisées pour les codes 404, 500 et 403.

Ce projet fournit ainsi une base solide et prête à l'emploi pour le développement de sites web et d'applications Django. Il permet aux développeurs de se concentrer sur la logique métier de leur projet plutôt que sur la configuration de l'environnement de développement.
